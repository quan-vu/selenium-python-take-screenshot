# Selenium Python - Take a screenshot of HTML Element

Required:

- Python 3.8
- Selenium
- Chrome Driver

## Quickstart

Create virtualenv

```bash
virtualenv -p python3 venv-3.8
source venv-3.8/bin/activate
```

Install dependencies

```bash
pip install --upgrade pip
pip install -r requirements.txt
```

Run application

```bash
python app.py
```
